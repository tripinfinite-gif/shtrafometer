# Инструкция для настройки DNS на reg.ru

## Контекст

Мне нужно настроить домен **shtrafometer.ru** на reg.ru, чтобы он указывал на хостинг Vercel.
Сайт уже задеплоен на Vercel по адресу: https://shtrafometer.vercel.app

## Что нужно сделать

### Шаг 1: Войти в панель управления reg.ru
- Зайти на reg.ru в раздел "Мои домены"
- Выбрать домен shtrafometer.ru
- Перейти в раздел "DNS-серверы и управление зоной" (или "DNS-управление")

### Шаг 2: Убедиться что используются DNS reg.ru
- DNS-серверы должны быть ns1.reg.ru и ns2.reg.ru (стандартные)
- Если стоят другие — переключить на DNS reg.ru

### Шаг 3: Настроить DNS-записи

**Удалить** все существующие A-записи и CNAME для @ и www (если есть).

**Добавить следующие записи:**

| Тип | Имя (хост) | Значение | TTL |
|-----|------------|----------|-----|
| A | @ | 76.76.21.21 | 3600 |
| CNAME | www | cname.vercel-dns.com. | 3600 |

Это стандартные DNS-записи для Vercel:
- A-запись `76.76.21.21` — это IP Vercel для apex-доменов (без www)
- CNAME `cname.vercel-dns.com` — для поддомена www

### Шаг 4: Проверить что нет конфликтов
- Убедиться что нет других A-записей для @ (кроме нашей)
- Убедиться что нет AAAA-записей для @ (если есть — удалить)
- Записи MX, TXT (SPF/DKIM) для почты — НЕ трогать, они не мешают

### Шаг 5: Сохранить изменения
- Применить/сохранить зону
- DNS-пропагация займёт от 5 минут до 48 часов (обычно 15-30 минут)

### Шаг 6: Проверка (после пропагации)
Можно проверить через:
- https://www.whatsmydns.net/#A/shtrafometer.ru — должен показать 76.76.21.21
- https://www.whatsmydns.net/#CNAME/www.shtrafometer.ru — должен показать cname.vercel-dns.com

## Дополнительно: Настройка почтовых записей (для email-рассылок)

Если будем использовать Resend для отправки email, нужно будет добавить:

| Тип | Имя (хост) | Значение | TTL |
|-----|------------|----------|-----|
| MX | send | feedback-smtp.us-east-1.amazonses.com | 3600 |
| TXT | resend._domainkey | (будет сгенерирован в Resend dashboard) | 3600 |
| TXT | @ | v=spf1 include:amazonses.com ~all | 3600 |

Точные значения DKIM/SPF будут известны после регистрации домена в Resend.

## Важно
- НЕ переключать домен на хостинг reg.ru — сайт живёт на Vercel
- НЕ удалять MX-записи если на домене уже есть почта
- SSL-сертификат Vercel выпустит автоматически после привязки домена
