# Инструкция для Claude в Chrome — настройка Бегет

Выполнять промпты последовательно. После каждого шага проверяй результат.

---

## Промпт 1: Заказать VPS на Бегете

```
Помоги мне заказать VPS-сервер на beget.com для хостинга Next.js приложения.

Контекст:
- Мне нужен VPS для Node.js приложения (Next.js 16, SSR)
- Нужен PostgreSQL для базы данных
- Домен: shtrafometer.ru
- Сервер должен быть в РОССИИ (это критично для 152-ФЗ о персональных данных)

Пошагово:

1. Зайти на beget.com → войти в аккаунт (или зарегистрироваться)
2. Перейти в раздел VPS / Виртуальные серверы / Cloud
3. Заказать VPS с параметрами:
   - ОС: Ubuntu 22.04 или 24.04
   - RAM: минимум 1 ГБ (рекомендую 2 ГБ)
   - CPU: 1-2 ядра
   - Диск: 10-20 ГБ SSD
   - Расположение: Россия (обязательно!)
4. После создания VPS — записать:
   - IP-адрес сервера
   - Логин root
   - Пароль (придёт на email или покажут в панели)
5. Убедиться что SSH-доступ работает (порт 22)

ВАЖНО: Нужен именно VPS, а НЕ обычный shared-хостинг. На shared-хостинге Бегета нет Node.js.

Если уже есть VPS — просто дай мне IP, логин и пароль.

Веди меня пошагово по интерфейсу Бегета.
```

## Промпт 2: Настроить SSH-ключ

```
Помоги мне настроить SSH-ключ для доступа к VPS на Бегете.

У меня есть VPS с данными:
- IP: [ВСТАВИТЬ IP]
- Логин: root
- Пароль: [ВСТАВИТЬ ПАРОЛЬ]

Мне нужно:

1. Зайти в панель управления Бегета
2. Найти раздел SSH-ключи (или настройки VPS → SSH)
3. Добавить SSH-ключ. Мой публичный ключ:

[Если ключ уже есть на Mac — выполни в терминале: cat ~/.ssh/id_rsa.pub и вставь сюда]

Если раздела с ключами нет в панели — это нормально, ключ можно добавить через SSH-команду:
ssh-copy-id root@[IP]

Также проверь что в панели VPS:
- Firewall разрешает порты: 22 (SSH), 80 (HTTP), 443 (HTTPS), 5432 (PostgreSQL только внутренний)
- ОС = Ubuntu 22.04 или 24.04

Веди меня пошагово.
```

## Промпт 3: Привязать домен к VPS

```
Помоги мне привязать домен shtrafometer.ru к VPS на Бегете.

Контекст:
- VPS IP: [ВСТАВИТЬ IP]
- Домен shtrafometer.ru зарегистрирован на reg.ru
- Сейчас DNS на reg.ru указывает на Vercel (старый хостинг), нужно перенаправить на Бегет

На reg.ru нужно изменить DNS-записи:

1. Зайти на reg.ru → Мои домены → shtrafometer.ru → DNS
2. Изменить A-запись для @:
   - Было: 216.198.79.1 (Vercel)
   - Новое: [IP VPS на Бегете]
3. Изменить CNAME для www:
   - Было: dd3fefdc6c646aed.vercel-dns-017.com.
   - Новое: удалить CNAME, добавить A-запись для www → [IP VPS на Бегете]
4. Сохранить

Также в панели Бегета:
5. Зайти в управление VPS → Домены
6. Привязать домен shtrafometer.ru к VPS (если есть такая опция)

DNS-пропагация займёт 15-30 минут.

НЕ ДЕЛАЙ ЭТО ПОКА Я НЕ ПОДТВЕРЖУ ЧТО VPS ГОТОВ И САЙТ НА НЁМ РАБОТАЕТ!
Сначала настроим VPS, потом переключим DNS. Иначе сайт ляжет.

Пока только подготовь инструкцию, что нужно будет изменить. Переключим DNS в самом конце.
```

## После получения SSH-данных

Как только дашь мне IP, логин и пароль от VPS — я сам через SSH:

1. Установлю Node.js 24, PostgreSQL 16, Nginx, Certbot
2. Создам базу данных и мигрирую данные из Neon
3. Заменю @vercel/postgres на стандартный pg-драйвер
4. Соберу и запущу приложение через PM2
5. Настрою Nginx как reverse proxy
6. Настрою SSL-сертификат (Let's Encrypt)
7. Настрою автозапуск и автообновление сертификатов

Сайт будет работать на VPS, все данные — в России.
