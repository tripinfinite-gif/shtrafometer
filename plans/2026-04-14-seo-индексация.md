### Обязательно до следующего деплоя

**1. Яндекс.Вебмастер — верификация**
```
1. webmaster.yandex.ru → Добавить сайт → https://shtrafometer.ru
2. Метод верификации: HTML-мета теги
3. Скопировать значение content="XXXXXXXX"
4. На сервере в .env.local добавить:
   NEXT_PUBLIC_YANDEX_WEBMASTER_KEY=XXXXXXXX
5. Задеплоить: git pull + docker compose up -d --build app
6. В Вебмастере нажать Проверить
7. Добавить sitemap: https://shtrafometer.ru/sitemap.xml
8. Указать регион: Москва
```

**2. og:image — создать файл**
```
Параметры: 1200×630 px, PNG
Дизайн: фиолетовый фон #6C5CE7, белый текст «Штрафометр», подпись снизу
Положить в: /public/og-image.png
```

**3. Задеплоить изменения кода**
