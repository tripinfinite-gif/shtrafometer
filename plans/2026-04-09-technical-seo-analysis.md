# План: Технический SEO-анализ в Штрафометре

**Дата:** 2026-04-09
**Статус:** planning

## Суть

Добавить в Штрафометр блок **технического SEO-анализа** сайта. Результат показывается на главной (краткая оценка) и подробно раскрывается в PDF-отчёте вместе с юридическим анализом.

## Что проверяем (технический SEO)

| # | Проверка | Категория |
|---|----------|-----------|
| 1 | SSL/HTTPS | Безопасность |
| 2 | Мета-теги (title, description, h1) | On-page |
| 3 | sitemap.xml — наличие и валидность | Индексация |
| 4 | robots.txt — наличие и корректность | Индексация |
| 5 | Schema.org (структурированные данные) | Разметка |
| 6 | Core Web Vitals (скорость загрузки) | Производительность |
| 7 | Битые ссылки (404) | Техническое |
| 8 | Редиректы (301/302 цепочки) | Техническое |
| 9 | Оптимизация изображений (alt, размер, формат) | On-page |
| 10 | Canonical URL | Индексация |
| 11 | Open Graph / соцсети мета-теги | On-page |
| 12 | Mobile-friendly (viewport, адаптивность) | UX |
| 13 | Favicon | UX |
| 14 | Язык страницы (lang атрибут) | Доступность |

## Вывод на главной странице

Краткий блок после юридического анализа:

- **Оценка** — цветовой индикатор (зелёный/жёлтый/красный) или балл из 100
- **Что проанализировано** — список из 14 пунктов с иконками (pass/warn/fail)
- **Что хорошо** — кратко, 2-3 пункта
- **Что плохо** — кратко, 2-3 пункта
- **Без рекомендаций** — для рекомендаций и решений → PDF-отчёт

## PDF-отчёт

Добавить раздел «Технический SEO-аудит» в существующий PDF:

1. Подробное описание каждой проблемы
2. Рекомендация по исправлению
3. Категория сложности: **быстрый фикс** или **сложный фикс**
4. Цена исправления (наша vs рынок)

## Ценовая логика

- **Наша цена = 30–50% от рынка** за счёт автоматизации через AI-агентов
- **Быстрые фиксы** включены в тариф «Автофикс» (уже есть)
- **Сложные фиксы** → отдельная услуга, обсуждается с клиентом
- **Комплекс (все быстрые + все сложные)** → скидка 30% от суммы
